# 🌌 GravityDistortionLab
A cinematic Python simulation of gravitational lensing around a supermassive black hole featuring Einstein rings, warped space-time grids, curved photon trajectories, animated magnification effects, and realistic light deflection visualizations using NumPy and Matplotlib.

---

✨ Features

- 🌀 Einstein Ring simulation
- 🌠 Background star distortion
- ⚫ Black hole visualization
- 🌌 Space-time warp grid
- 💫 Curved photon trajectories
- 🔥 Cinematic heatmaps
- 🎞 Animated lensing effects
- 📈 Magnification visualization
- 🌌 Realistic gravitational deflection

---

🧠 Physics Behind It

The simulation is based on gravitational lensing equations derived from General Relativity.

Massive objects curve space-time and bend the path of light rays traveling nearby.

Main deflection equation:

α = (4GM) / (c²r)

Where:

- α = deflection angle
- G = gravitational constant
- M = mass of the lensing body
- c = speed of light
- r = radial distance from the lens
